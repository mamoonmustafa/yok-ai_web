Getting Started with Yok-AI

System Requirements

Before installing Yok-AI, ensure your system meets the following requirements:

  • Operating System: Windows 10 or Windows 11
  • Memory: Minimum 4GB RAM
  • Storage: 100MB free disk space
  • Internet Connection: Required for API communication

Installation Guide

  1. Download the Yok-AI installer package from our download page.
  2. Run the installer file YokAI_Setup.exe with administrator privileges.
  3. Follow the on-screen installation wizard:
    • Accept the End User License Agreement
    • Choose installation location (default: C:\Program Files\YokAI)
    • Select start menu folder and desktop shortcut options
    • Click "Install" to begin installation
  4. Wait for the installation to complete.
  5. Click "Finish" to exit the installer.

First-Time Setup

  1. Launch Yok-AI from the Start menu or desktop shortcut.
  2. On first launch, you'll need to enter API keys:
    • Claude API key (required): Obtain from Anthropic
    • Whisper API key (required for voice features): Obtain from OpenAI
  3. Keys are securely encrypted and stored locally on your system.
  4. The application will automatically validate your API keys.
  5. Once validated, Yok-AI is ready for use!

Tip: If you don't have API keys yet, you can sign up for them at the respective provider websites. For Claude API access, you may need to apply for their API program if you're not already a part of it.

Core Features

Interface Overview

The Yok-AI interface consists of several key components:

  • Title Bar: Window controls, context settings, and help menu
  • Control Panel: Screenshot and voice capture buttons
  • Chat History: Displays conversation and responses
  • Input Area: Text entry for queries and commands
  • Status Bar: Shows session time and usage counters

Text Interaction

  1. Type your question or request in the input box at the bottom.
  2. Press Ctrl+Enter or click the send button (âž¤).
  3. Wait for AI response to appear in the chat history.
  4. You can send up to 100 text messages per session.

Screenshot Analysis

Capture and analyze screen content with our AI:

  1. Press Ctrl+Alt+C or click the "Screen Capture" button.
  2. The application will briefly hide itself and capture your screen.
  3. You can capture up to 3 screenshots per analysis session.
  4. After capture, the application enters screenshot mode:
    • Add custom instructions in the text field (optional)
    • Click "Clear" to discard screenshots and start over
    • Click "Analyze All" to process the screenshots
  5. The AI will analyze the content and provide a response.

Tip: For best results with complex screenshots, add specific instructions like "Explain the code in this screenshot" or "Summarize the key data points from this graph."

Voice Capture

Capture and analyze spoken audio:

Method 1: Press and hold the "Voice Capture" button

  • Speak clearly during recording
  • Release the button when finished to process audio

Method 2: Press Ctrl+Alt+V to toggle recording mode

  • Start recording with first press
  • Press again to stop recording and process audio

The application will transcribe the audio and generate a response. Maximum 50 audio transcriptions per session.

Tip: For better audio recognition, speak clearly and try to minimize background noise. Recordings should be at least 3 seconds long for optimal processing.

Meeting Context Management

  1. Click the context button (ðŸ“‹) in the title bar.
  2. In the context panel:
    • Enter meeting information, notes, or context in the text field
    • Add images (up to 3) using the "Add Image" button
    • Click "Generate Summary" to create a context summary
    • Review the summary and click "Save Context"
  3. Once saved, all AI responses will consider this context.
  4. The context indicator will show when context is active.

Limited to 2 context changes per session.

Troubleshooting

API Connection Issues

If you encounter Error Code EA22578 (Invalid or missing API key):

  • Verify API keys are entered correctly
  • Check your internet connection
  • Ensure API keys have sufficient credits

Capture Functionality

If screen capture fails:

  • Check for conflicting screen recording software
  • Ensure application has necessary permissions
  • Restart the application

Audio Processing

If voice capture doesn't work:

  • Check microphone access permissions in Windows
  • Ensure no other application is using the microphone
  • Try recording for at least 3 seconds for better processing

Performance Issues

If responses are slow:

  • Check your internet connection speed
  • Reduce the number of attached images or screenshot complexity
  • Close other resource-intensive applications

Emergency Reset

If the application becomes unresponsive:

  1. Press Ctrl+Alt+X to force close
  2. Delete temporary files from %LOCALAPPDATA%\StealthAssist (optional)
  3. Restart the application

Privacy & Security

Stealth Features

Yok-AI includes several privacy-enhancing features:

  • Stealth Operation: Operates invisibly with minimal system footprint
  • Secure Communications: End-to-end encrypted connection to Claude AI
  • Screen Capture Exclusion: Self-excludes from screenshots to maintain discretion
  • Keystroke Protection: Optional keystroke hiding protection against potential monitoring software

Visibility Controls

Yok-AI offers multiple visibility modes:

  1. Normal Mode: Standard window size and opacity
  2. Minimized Mode: Press Ctrl+Alt+Space to shrink window to 20x20 pixels with reduced opacity
  3. Hidden Mode: Close the window using the X button (application continues running in background)
  4. Keystroke Protection: Press Ctrl+Alt+K to activate keystroke hiding (prevents keyloggers)

Data Privacy

Yok-AI respects your privacy:

  • User content is primarily stored locally on your device
  • Cloud-processed data is not permanently stored after processing unless necessary for service provision
  • API Key Management: Secure encryption of API credentials
  • Anti-Detection Mechanisms: Multiple layers of protection against detection
  • Temporary File Management: Automatic cleanup of temporary files

Tip: For maximum privacy, use the keystroke protection feature when entering sensitive information.

Frequently Asked Questions

How does Yok-AI remain invisible during screen sharing?

Yok-AI uses special rendering technology that makes it completely invisible during screen sharing and recordings. It detects when screen sharing is active and automatically adjusts its rendering layer to ensure it doesn't appear in shared content.

Can I use Yok-AI on macOS or Linux?

Currently, Yok-AI is only available for Windows 10 and Windows 11. We're evaluating support for other platforms in future releases.

Do I need to be connected to the internet to use Yok-AI?

Yes, Yok-AI requires an internet connection to communicate with Claude AI and Whisper API services.

How secure is my data with Yok-AI?

Yok-AI implements industry-standard security measures including end-to-end encryption for data transmission. Your content is primarily processed locally when possible, and cloud-processed data is not permanently stored after processing unless necessary for service provision.

What happens if I run out of credits?

When you run out of credits, you'll need to wait until your next billing cycle for your credits to reset, or upgrade to a higher tier plan with more credits.

How do I uninstall Yok-AI?

To uninstall Yok-AI:

  1. Open Windows Settings > Apps > Apps & features
  2. Search for "Yok-AI"
  3. Click Uninstall and follow the prompts
  4. Optional: Delete remaining files from %LOCALAPPDATA%\StealthAssist
